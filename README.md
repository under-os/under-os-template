# UnderOS Default Template

Install

```
mkdir -p ~/Library/RubyMotion/template/uos
git clone https://github.com/under-os/under-os-template.git ~/Library/RubyMotion/template/uos
```

Use

```
motion create awesomness --template=uos
```
